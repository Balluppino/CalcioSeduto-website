## Project Overview

**Calcio Seduto** is a modern, static website created to present and promote an inclusive sport for children with motor disabilities in Milan. The site is fully **responsive**, optimized for SEO (meta tags, Open Graph) and uses **Schema.org** structured data to improve discoverability.

## Site Structure

| Section | Description |
|--------:|-------------|
| **Header & Navigation** | Accessible nav with hamburger menu for mobile and links to main pages. |
| **Hero** | Full-width background image with title and primary CTA: “Come sit with us!”. |
| **About / Who We Are** | Emotional story of the project and founders. |
| **Official Rules** | Link to the official seated-football rules (internal anchor on About page). |
| **Gallery** | Photo gallery showcasing training sessions and events. |
| **Sign Up** | Page for new registrations and interest forms. |
| **Press / Testimonials** | Articles, interviews and mentions about the project. |
| **Contact** | Contact information, phone and emails. |
| **Footer** | Partners, useful links and copyright notice. |

## Implemented Features

- **Animated preloader** featuring the official logo.
- **Responsive image slider** with navigation arrows and dots.
- **SEO-ready meta tags** (title, description, canonical) and **Open Graph** tags.
- **Schema.org JSON-LD** block for organization markup.
- **Mobile-first navigation** with accessible ARIA attributes.
- **Clickable contact links** (`tel:` and `mailto:`) for immediate user action.
- **Favicons** and web manifest for cross-device support.

## Project Goals

- Increase visibility for the Calcio Seduto initiative.
- Make it easy for families to find training info and sign up.
- Share the story behind the team and promote sports inclusion.
- Provide a clean, accessible web presence for partners and sponsors.
